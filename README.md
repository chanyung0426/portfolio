# REACT - Mcdonald Project

리액트 맥도날드 사이트

## 🖥️ 프로젝트 소개

맥도날드 사이트를 참고하여 개발한 사이트입니다.

## 🕰️ 개발 기간

23.12.04 - 24.02.24

### ⚙️ 개발 환경

- `Javascript`
- `Html`
- `css`
- **Framework** : React
- **Database** : Firebase

## 📌 주요 기능

- 메뉴 등록 페이지 제작(관리자 권한)
- 메뉴 등록, Firebase Database에 저장
- 검색창 : 검색 시 메뉴 별 알레르기 정보 구현
- 카테고리별 리스트 제작
- 메뉴 상세페이지 제작
- REST API: Kakao Map API를 활용한 매장 위치 정보 구현
