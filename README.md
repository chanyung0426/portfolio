맥도날드 사이트를 참고하여 개발한 사이트입니다.

## 🕰️ 개발 기간

### ⚙️ 개발 환경

- `Javascript`
- `HTML`
- `CSS`
- **Framework** : React
- **Database** : Firebase
- **REST API** : Kako Map API

- 메뉴 등록 Firebase Database 저장
- 검색창 구현 : 검색 시 메뉴 별 알레르기 정보 구현
- 카테고리별 리스트 제작
- 메뉴 상세페이지 제작
